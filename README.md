You can run this project in two different ways (pick one):

1) Using yarn

Execute the command below at the root of the folder:

```
  $ yarn && yarn start
```

2) Using docker

Execute the command below at the root of the folder:

```
  $ docker-compose up
```

After starting the frontend, access the project at:

```
  http://localhost:3000
```
